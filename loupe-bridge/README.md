# Contao Loupe bridge

[![](https://img.shields.io/packagist/v/contao/loupe-bridge.svg?style=flat-square)](https://packagist.org/packages/contao/loupe-bridge)
[![](https://img.shields.io/packagist/dt/contao/loupe-bridge.svg?style=flat-square)](https://packagist.org/packages/contao/loupe-bridge)

The Loupe bridge provides a pre-configured bridge between Contao, [SEAL][1] and [Loupe][2].

Contao is an Open Source PHP Content Management System for people who want a professional website that is easy to
maintain. Visit the [project website][3] for more information.

## Other adapters

If you want to use a different search adapter, do not install this package but your desired [SEAL adapter][4] directly
instead.
